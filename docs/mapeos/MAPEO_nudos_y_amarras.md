## Requisito DSA 3: Diferencia cuerda estática vs dinámica + 3 usos c/u
**Estado:** ⏳ Pendiente
**Ubicación en manual GO!:** Sección "¿CONOCES? Tipos de cuerdas"
**Conceptos a incluir:**
- **Estáticas:** Casi no se elastecen (2% máx), fibras paralelas + manga, no torsión, para rescate/seguridad, puentes, tirolinas (9-12mm). NO para escalada/aseguramiento.
- **Semi-estáticas:** Poliamida, resisten desgaste, para trabajos verticales
- **Dinámicas:** Actúan como amortiguador, gran elasticidad, fibras trenzadas helicoidalmente, alta absorción impactos, para escalada (7.9-11mm). Tipos: simples, dobles, gemelas.

**Usos estáticas:** Puentes de cuerda, tirolinas, trabajos verticales
**Usos dinámicas:** Escalada deportiva, montañismo, rescates con caídas

## Requisito DSA 4: Identificar tipos de sogas (poliéster, sisal, nylon, polipropileno)
**Estado:** ⏳ Pendiente
**Ubicación en manual GO!:** Sección "MATERIALES - ¿De qué está hecha tu cuerda?"
**Conceptos a incluir:**
- **Poliéster:** Gran resistencia, compatible con otras fibras, para estáticas y capa de dinámicas
- **Sisal:** Resistente, durable, estirable, para marina/agricultura, fibras lisas amarillas
- **Nylon:** Superficie lisa/sedosa, enrosca fácil, no flota, fibra sintética, blanco o colores
- **Polipropileno:** Amarillo generalmente, flota en agua, rígido/quebradizo, marino

## Requisito DSA 5: Ventajas/desventajas cuerda sintética + tipos de fibras
**Estado:** ⏳ Pendiente
**Ubicación en manual GO!:** Integrado en sección "MATERIALES"
**Conceptos a incluir:**
- **Ventajas sintéticas:** Mayor resistencia y durabilidad que naturales
- **Desventajas sintéticas:**
  * Nylon: pierde 5% fuerza mojado
  * Polietileno: poco resistente a abrasión
  * Poliéster: no flota
- **Fibras naturales vs sintéticas:** Naturales se deterioran por descomposición/moho, NO para aplicaciones críticas
